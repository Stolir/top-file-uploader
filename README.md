# File Uploader

A lightweight file storage app built as a practice project for The Odin Project Course.

## Features

- **Authentication** — Secure sign-up and login with session-based auth via Passport.js (Local Strategy)
- **File Uploads** — Upload files to your personal storage space
- **Folder Management** — Create, rename, and delete folders to organize your files
- **File Operations** — View, download, and delete uploaded files
- **Private Storage** — Each user only sees their own files and folders
- **Persistent Sessions** — Stay logged in across page refreshes

---

## Tech Stack

| Layer         | Technology                   |
| ------------- | ---------------------------- |
| Runtime       | Node.js                      |
| Framework     | Express.js                   |
| Templating    | EJS                          |
| Auth          | Passport.js (Local Strategy) |
| ORM           | Prisma                       |
| Database      | PostgreSQL                   |
| Session Store | express-session              |
